# GCNvsGAT
In this project we will try to implement manually the GCNs and GATs and compare them on standard datasets (Cora,Citeseer, PubMed)

Download the cora dataset in the following link:
https://graphsandnetworks.com/the-cora-dataset/

Download the Citesser dataset in the following link:
https://github.com/ZPowerZ/citeseer-dataset

Download the PubMed dataset using the download_pubmed.py file.
